# IMU Logger State Machine

## 상태 목록

```
BOOT
SELF_TEST
READY
CALIBRATION
RECORDING
FLUSHING
WIFI_CONFIG
USB_MSC
ERROR
SLEEP
LOW_BATTERY_PROTECT
```

### BOOT

```
hardware init
config load
```

### SELF_TEST

```
IMU check
SD mount
RTC check
```

### READY

가능 기능

```
record start
config change
wifi
usb
calibration
```

### RECORDING

```
IMU sampling
marker insert
segment rotate
```

금지

```
USB_MSC
SD format
```

### FLUSHING

```
drain buffers
write footer
close file
```

### WIFI_CONFIG

```
start AP
web config
file download
```

### USB_MSC

```
SD expose to PC
internal write disabled
```

### ERROR

```
store error code
LED error
```

### SLEEP

```
low power
motion wake
```

### LOW_BATTERY_PROTECT

```
safe flush
shutdown prepare
```

### 주요 전이

```
BOOT → SELF_TEST → READY
READY → RECORDING → FLUSHING → READY
READY → WIFI_CONFIG → READY
READY → USB_MSC → READY
READY → SLEEP → READY
```
