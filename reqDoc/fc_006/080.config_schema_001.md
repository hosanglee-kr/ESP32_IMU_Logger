# config_schema_001.md

# IMU Logger Configuration Schema Specification

## 1 목적

LittleFS 기반 JSON 설정 구조 정의

저장 파일

```
/config.json
/config.bak
/runtime.json
/calib.json
/stats.json
```

### config.json 구조

```json
{
  "schema_version":1,
  "device":{},
  "logging":{},
  "imu":{},
  "marker":{},
  "power":{},
  "ui":{},
  "wifi":{},
  "time":{},
  "usb":{},
  "diagnostics":{},
  "file_policy":{}
}
```

### logging 예

```json
"logging":{
 "sample_rate_hz":1000,
 "segment_minutes":5,
 "prealloc_mb":64,
 "sync_interval_sec":5,
 "checkpoint_interval_sec":5
}
```

### imu 예

```json
"imu":{
 "gyro_range_dps":2000,
 "accel_range_g":16,
 "filter_profile":"gyroflow_hq",
 "fifo_enabled":true,
 "auto_calibration":true,
 "orientation":"top_mount"
}
```

### marker 예

```json
"marker":{
 "impact_detect_enabled":true,
 "impact_threshold_g":2.5,
 "gyro_threshold_dps":350,
 "freefall_threshold_g":0.25
}
```

### runtime.json 예

```json
{
 "schema_version":1,
 "last_mode":"READY",
 "last_session_id":42
}
```

### calib.json 예

```json
{
 "gyro_bias":{"x":12,"y":-8,"z":5},
 "valid":true
}
```

### stats.json 예

```json
{
 "boot_count":120,
 "unclean_shutdown_count":3,
 "max_sd_latency_us":84500
}
```

핵심 원칙

* validation 필수
* atomic save
* migration 지원
