# config_schema_001.md

# IMU Logger Configuration Schema Specification

## 1 목적

LittleFS 기반 JSON 설정 구조 정의

저장 파일

```
/config_001.json
/config_001.bak
/runtime_001.json
/calib_001.json
/stats_001.json
```

### config_001.json 구조

```json
{
  "meta": {
    "fileName": "config_001.json",
    "schema_ver": 1,
    "last_updated": "2026-03-09T23:19:29Z"
  },
  "config": {
    "device":{},
    "logging":{},
    "imu":{},
    "marker":{},
    "power":{},
    "ui":{},
    "wifi":{},
    "time":{},
    "usb":{},
    "diagnostics":{},
    "file_policy":{}
  }
}
```
