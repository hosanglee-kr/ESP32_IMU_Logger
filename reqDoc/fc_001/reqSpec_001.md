# NiYien Tool 및 Gyroflow 소개

## 1. Gyroflow 소개

* **정의:** Gyroflow는 오픈소스 비디오 안정화 소프트웨어로, 자이로스코프 및 가속도계 데이터를 사용하여 정밀한 비디오 안정화, 젤리 현상 제거, 렌즈 왜곡 보정을 수행합니다.
* **장점:** 충분한 조명 환경에서는 짐벌과 필적하는 안정화 효과를 제공합니다. DaVinci, PR/AE 플러그인을 통해 무손실 품질 안정화도 가능합니다. NiYien Tool은 사용 편의성을 간소화합니다.
* **자이로 기반 안정화의 한계:**
    * 셔터 속도 1/120 이상 권장. (빠른 움직임일수록 높은 셔터 속도 필요)
    * 초점 거리 150mm 이하 권장.
    * 단일 비디오 클립 내에서 줌 기능 미지원.

## 2. NiYien Tool 장치 소개 및 기본 조작

### 2.1 장치 구성 요소

1.  상태 표시등
2.  OLED 디스플레이 화면
3.  충전 및 데이터 인터페이스
4.  셔터 케이블 인터페이스
5.  작동 버튼
6.  메모리 카드 슬롯

### 2.2 기본 조작

#### 2.2.1 버튼 조작

* **<, > 버튼:** 좌/우 또는 상/하 이동
* **O 버튼:** 단일 클릭 (확인, 이전 인터페이스로 돌아가기), 길게 누르기 (켜기/끄기, 메인 인터페이스에서 끄기)
* **AF/S 버튼:** 2단계 버튼 (반누름: 초점, 전체 누름: 셔터/녹화)
* **자동 종료:** 15분간 활동이 없으면 자동 종료

#### 2.2.2 기능 및 인터페이스 표시

* **메인 인터페이스:** Sensflow, TimeLapse, Shutter, Inclino, Settings
* **상태 표시줄:** 현재 시간 (좌측 상단), 메모리 카드 및 배터리 표시 (우측 상단)
* **Senseflow:** 자이로스코프 녹화 시간, 렌즈 번호, 초점 거리, 남은 녹화 시간 표시
* **TimeLapse:** 대기, 노출, 간격 시간, 촬영 샷 수 설정 가능. O 버튼으로 모드 진입/수정.
* **Shutter:** 수동 셔터 모드 또는 벌브 모드 선택 가능.
* **Inclino:** X (피치), Y (롤) 각도 동적 표시.
* **Settings:** 언어, 자이로 보정, SD 포맷 옵션 제공.

### 2.4 자이로스코프 보정

* **필요성:** 3개월 이상 미사용 시 최적의 안정화 정확도를 위해 보정 권장.
* **방법:** Settings에서 Gyro calibration 선택 후 평평한 곳에 장치 고정. 실패 시 재시도.
* **참고:** 공장 출고 시 이미 보정되어 있어 바로 사용 가능.

### 2.5 메모리 카드 읽기 및 포맷

* **데이터 읽기:** Type-C 케이블로 연결하여 USB 드라이브로 인식 후 데이터 직접 내보내기. 대용량 데이터는 카드 리더기 사용.
* **포맷:** Settings에서 Format SD 선택.
* **주의:** PC에서 포맷 시 오류 발생 가능성이 있으므로 장치에서 포맷 권장.

### 2.6 펌웨어 업그레이드

* 세부 내용은 섹션 5.6.3 펌웨어 업그레이드 참조.

## 3. 카메라 동기화

### 3.1 카메라 설정

* 카메라에서 바디 내 안정화 및 렌즈 안정화 기능 비활성화.

### 3.2 시간 동기화 조작

* **방법:** 카메라 녹화 버튼과 자이로스코프의 AF/S 버튼을 동시에 누름 (지속 시간 10초 미만).
* **반복:** 다시 한번 카메라 녹화 버튼과 자이로스코프 AF/S 버튼을 동시에 누름.
* **결과:** 자이로스코프는 계속 녹화되며, 카메라는 평소처럼 사용.
* **주의:** 이 조작은 하루에 한 번만 필요하며, 한 번 동기화하면 그날 다시 할 필요 없음.
* **참고:** 자이로스코프 데이터 용량이 크므로 카드 리더기 사용을 권장.
