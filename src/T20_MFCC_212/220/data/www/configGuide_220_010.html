1. 제품 소개 (Introduction)

T20 MFCC v220은 산업 현장의 회전 기기(모터, 펌프, 베어링 등)의 상태를 실시간으로 감시하고, 이상 징후를 포착하여 엣지 레벨에서 인공지능 특징량(MFCC)을 추출하는 고성능 분석 장치입니다.

핵심 가치: 단순한 데이터 로깅을 넘어 사고 직전 4초의 징후를 포착하는 "산업용 블랙박스" 기능과 TinyML 학습을 위한 "데이터 자동 생성기" 역할을 수행합니다.

2. 시스템 사양 (Hardware Specs)

메인 프로세서

MCU: 32-bit Dual-Core (240MHz)
AI 가속: SIMD(벡터병렬처리) 명령어 지원
메모리: 내장 512KB SRAM, 외장 8MB PSRAM 탑재

정밀 센서

Sensor: Bosch BMI270 IMU
사양: 6-axis 16-bit 가속도/자이로스코프

저장 및 통신

저장: SD_MMC (Zero-Copy DMA)
네트워크: Wi-Fi 4 (802.11 b/g/n), MQTT 3.1.1, WebSocket
업데이트: OTA 무선 업데이트 지원

3. 신호처리 아키텍처 (Architecture)

데이터의 무손실 확보와 정밀 연산을 위해 듀얼 코어가 독립적으로 작동하는 이원화 구조를 가집니다.

Core 0: 무손실 수집 엔진

1
FIFO Batch Read: 센서 내부 버퍼에서 1600Hz 데이터를 10ms 단위로 퍼올려 샘플 누락을 0%로 유지합니다.
2
PSRAM 핑퐁 버퍼: 8MB PSRAM에 3축 데이터를 분리 적재하여 대용량 프리-트리거 링버퍼의 기반을 마련합니다.

Core 1: 고속 분석 및 저장 엔진

3
시간 도메인 전처리: DC Offset 제거로 중력을 배제하고, Pre-emphasis 필터로 고주파 마찰음을 증폭합니다.
4
주파수 변환 (FFT): SIMD 가속을 통해 256~4096 크기의 FFT 연산을 수행하여 파워 스펙트럼을 산출합니다.
5
스펙트럼 감산 (Noise Cancel): 고정/적응형 알고리즘을 통해 주변 소음을 지우고 순수 이벤트만 남깁니다.
6
MFCC 특징량 추출: Mel-Filterbank와 DCT-II를 거쳐 진동의 음색(Timbre)을 39차원 벡터로 압축합니다.
7
블랙박스 Flush: 트리거 발생 시 PSRAM에 담겨있던 "과거 4초 이력"을 SD카드 파일 최상단에 즉시 플러시합니다.

4. 작동 및 사용 방법 (Operation)

웹 대시보드 접속

  • AP 접속: Wi-Fi 목록에서 'T20_MFCC_AP' 선택 (암호: 12345678)
  • 주소창: http://192.168.4.1 접속

핵심 버튼 기능

  • START/STOP: 수동 계측 및 SD카드 기록 제어
  • NOISE LEARN: 현재 소음을 학습하여 캔슬링 기준(Profile)으로 설정
  • SENSOR CALIBRATE: 장비 설치 상태에 따른 영점 조절 (Z축을 하늘로 향하게 배치)

6. 산업 현장 최적화 가이드 (Tuning Guide)

💡 케이스 1: 배터리 구동 간헐적 감시

목표: 전기를 아끼면서 모터 가동 시만 기록
설정: Deep Sleep ON, Wake Thres 0.5G, Pre-Trigger 4s.
결과: 평소 수면하다가 충격 발생 시 깨어나 사고 4초 전부터 자동 저장.


💡 케이스 2: 고주파 마찰음 이상 탐지

목표: 베어링의 미세한 '끼익' 소리만 잡고 싶을 때
설정: Pre-emphasis 0.98, Smart Trigger Band 500-800Hz 설정.
결과: 큰 진동에는 반응하지 않고 오직 특정 주파수 에너지가 튈 때만 레코딩.


💡 케이스 3: TinyML 모델 학습용 데이터 수집

목표: 파이썬 AI 학습을 위한 고밀도 데이터 확보
설정: Payload 'Tensor', Save Raw Wave 'ON', Rotation 5MB.
결과: 16프레임씩 묶인 특징량과 원본 파형을 동시에 수집하여 즉시 학습 가능.
