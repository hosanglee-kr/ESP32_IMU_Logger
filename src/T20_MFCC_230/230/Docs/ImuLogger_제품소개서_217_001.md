# 🚀 제품 소개서: SMEA (Super Micro Edge AI)

## "기계/제품의 미세한 비명 소리, 이제 담배값크기의 SMEA내의 AI가 듣고 판단해서 이상상황을 알려줍니다."

공장 설비가 갑자기 멈춰서 막대한 손해를 본 적이 있으신가요?
불량품을 걸러내기 위해 아직도 사람의 감각에만 의존하고 계신가요?

이제, 당신의 현장에 **SMEA (Super Micro Edge AI)**를 도입할 때입니다.

### 🔍 SMEA란 무엇인가요?
SMEA는 설비의 진동과 움직임을 1초에 1,600번씩 정밀하게 측정하고, 기기 스스로 정상 상태와 이상 상태를 구분해내는 **'초소형 인공지능 예지보전 솔루션'**입니다.

### 💡 왜 기존 방식이 아닌 SMEA여야 할까요?

**1. 💻 비싸고 거대한 PC는 이제 필요 없습니다.**
기존에는 진동을 분석하기 위해 설비 옆에 커다란 산업용 PC를 설치하고, 복잡한 선을 연결해야 했습니다. SMEA는 **동전 두 개 크기**의 초소형 칩 안에서 이 모든 과정을 독립적으로 수행합니다. 가격은 기존 PC 시스템의 **수십 분의 일**에 불과하며, 좁은 틈새나 회전하는 기계 부품 위에도 찰딱 붙일 수 있습니다.

**2. 🌐 서버도, 전용 어플리케이션도 필요 없는 All-in-one**
매달 고액의 클라우드 서버 비용을 내거나, 무거운 전용 프로그램을 PC나 스마트폰에 설치할 필요가 없습니다. SMEA 자체에 **초소형 웹 서버가 내장**되어 있습니다. 스마트폰이나 태블릿에서 와이파이만 연결하면, 웹 브라우저를 통해 실시간 상태를 보고 즉시 설정을 변경할 수 있습니다.
* **[측정 ➔ 노이즈 제거 ➔ 특징 분석 ➔ 이상 감지]** 이 모든 것이 칩 하나에서 끝납니다.

**3. 🧠 나만의 기계를 이해하는 맞춤형 AI**
기계마다 떨림의 특징은 모두 다릅니다. SMEA의 **'노이즈 학습(Noise Learn)'** 버튼을 누르기만 하면, AI가 현재 기계의 정상적인 작동 상태를 스스로 학습하고 기억합니다. 이후 평소와 다른 미세한 불량 진동이나 고장 전조 증상이 발생하면 즉각적으로 감지해냅니다.

### 🎯 어디에 사용할 수 있나요?
* **예지보전 (설비 지킴이):** 모터, 펌프, 컨베이어 벨트 등 핵심 부품에 부착하여 고장 전조 증상을 미리 파악하고 돌발 정지를 막습니다.
* **품질 검사 (불량 감별사):** 생산 라인에서 조립 불량, 나사 풀림, 내부 균열 등에서 발생하는 미세한 이음/진동을 감지하여 불량품을 자동으로 솎아냅니다.

# 📖 SMEA 사용자 매뉴얼

SMEA는 누구나 쉽게 사용할 수 있도록 직관적인 화면을 제공합니다. 기기의 전원을 켜고 스마트폰이나 PC로 SMEA의 와이파이(예: `T20_MFCC_AP`)에 접속한 뒤, 인터넷 창에 `192.168.4.1`을 입력하세요.

## 1. Dashboard (실시간 현황판)
기계의 현재 상태를 한눈에 보는 메인 화면입니다.

* **START / STOP (측정 시작/정지):** 버튼 하나로 데이터 수집과 인공지능 분석을 시작하고 멈춥니다.
* **NOISE LEARN (정상 상태 학습):** 기계가 가장 부드럽게 잘 돌아가고 있을 때 이 버튼을 누르세요. 기기가 현재의 진동을 '정상'으로 기억하고, 불필요한 주변 소음이나 기본 떨림을 분석에서 제외합니다.
* **실시간 차트:** * *파형 차트:* 현재 기계가 얼마나 크게 떨리고 있는지 직관적으로 보여줍니다.
  * *지문 차트(MFCC):* 진동을 인공지능이 이해하기 쉬운 39개의 '진동 지문'으로 변환하여 보여줍니다. 고장이 발생하면 이 지문의 모양이 확연하게 달라집니다.

## 2. Settings (환경 설정)
우리 현장과 기계에 딱 맞게 기기를 세팅하는 곳입니다. (전문가가 아니어도 쉽게 설정할 수 있습니다.)

* **Output Mode (AI 출력 모드):**
  * *Single Vector:* 단순히 진동 데이터를 실시간으로 모니터링하고 저장할 때 사용합니다.
  * *Sequence Tensor:* 향후 기기 스스로 불량 여부를 최종 판독하는 엣지 AI(TinyML)를 가동할 때 사용하는 고급 모드입니다.
* **Sensor Configuration (센서 감도 설정):**
  * 기계의 떨림이 매우 크다면 측정 범위를 넓게(예: ±16G), 미세한 떨림을 잡아야 한다면 좁게(예: ±2G) 설정하세요.
* **Filter Cutoff (잡음 제거 필터):**
  * 기계의 본질적인 진동이 아닌, 에어컨이나 주변 차량 이동으로 인한 쓸데없는 웅웅거림(저주파)을 걸러내는 수치입니다.
* **Wi-Fi Configuration (통신 설정):**
  * 현장의 공유기 이름과 비밀번호를 입력해두면, 직접 접속하지 않고도 사무실의 PC에서 기기 상태를 확인할 수 있습니다.

## 3. Explorer (데이터 탐색기)
과거의 기록을 확인하고 다운로드하는 곳입니다.

* 기계에 문제가 생겼을 때, 기기 내부 메모리(SD카드 등)에 자동 저장된 진동 기록 파일을 PC로 다운로드하여 정밀 분석을 의뢰하거나 인공지능 학습 자료로 사용할 수 있습니다.

## 4. Diagnostics (기기 진단)
기기가 제대로 작동하고 있는지 확인합니다.

* **Sensor Calibration (영점 조절):** 기기를 처음 기계에 단단히 부착한 후, 기계가 멈춰있는 상태에서 한 번 눌러주세요. 기울어짐을 스스로 보정하여 더욱 정확한 측정이 가능해집니다.
