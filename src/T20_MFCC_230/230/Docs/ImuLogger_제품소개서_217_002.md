### **세상에서 가장 작고 똑똑한 AI 설비 주치의, SMEA-100 (Super Micro Edge AI)**

설비 예지보전과 품질 검사, 도입하고 싶지만 항상 **'비용'**이 문제셨죠?
센서 달고, 거대한 PC 사고, 복잡한 서버 구축하고, 유지보수 비용까지... 배보다 배꼽이 더 커서 포기했던 수많은 순간들.

### 🎯 **왜 SMEA-100을 선택해야 할까요?**

#### **1. 압도적인 가성비, 수백만 원대 시스템을 '단돈 10달러'로!**
수백, 수천만 원을 호가하던 무겁고 비싼 PC 기반 시스템은 잊으세요. SMEA-100은 담뱃갑만 한 초소형 크기에 불과하지만, 그 안에는 거대한 공장을 지키는 강력한 힘이 숨어있습니다. 수백 개의 설비에 전부 도입해도 부담 없는 가격! 진정한 의미의 **ROI(투자 대비 수익률) 혁명**을 경험하세요.

#### **2. "어? 소리가 이상한데?" 숙련된 장인의 귀를 완벽히 모방하다**
기계가 아프면 가장 먼저 '소리'와 '진동'이 달라집니다. SMEA-100은 수십 년 경력의 베테랑 작업자가 소리만 듣고도 기계의 이상을 알아채는 **'인간의 청각 인지 방식'**을 알고리즘으로 그대로 구현했습니다. 미세한 쇳소리, 불규칙한 떨림까지 AI가 놓치지 않고 잡아냅니다.

#### **3. No 서버! No 앱! 100% 완전 독립형 All-in-one**
서버 구축비용 제로! 무거운 어플리케이션 설치 제로!
SMEA-100은 측정 센서부터 데이터 저장, 분석, AI 이상 감지까지 이 작은 기기 하나에서 **전부 스스로 처리**합니다. 확인이 필요할 땐 스마트폰이나 PC의 '인터넷 브라우저'로 기기에 직접 접속하기만 하세요.

#### **4. 쓰레기 데이터는 버려라! 똑똑한 '알짜배기' 압축 저장**
매일 쏟아지는 방대한 정상 데이터를 저장하느라 비싼 클라우드 용량을 낭비할 필요가 없습니다. SMEA-100은 **"진짜 문제가 발생한 이상 데이터"**만 쏙쏙 골라 논리적으로 압축하고 저장합니다. 데이터 관리의 피로도와 비용을 획기적으로 줄여줍니다.

#### **5. 혼자서도 잘하지만, 같이 쓰면 더 좋은 API 연동**
기존에 사용 중인 사내 시스템(ERP, MES 등)이 있으신가요? SMEA-100은 단독으로도 완벽하게 작동하지만, 원하신다면 기존 시스템과 유연하게 연결(API 연동)할 수 있습니다.

# 📖 2. 사용자 기능 매뉴얼: SMEA-100 아키텍처 및 상세 기능

SMEA-100은 복잡한 과정을 거치지 않고, 기기 자체에서 모든 데이터 처리와 AI 분석이 이루어지는 **'엣지 컴퓨팅(Edge Computing)'** 아키텍처를 채택하고 있습니다.

아래는 SMEA-100이 소리를 듣고 문제를 판단하여 사용자에게 알려주기까지의 상세 처리 단계입니다.

---

## **⚙️ SMEA-100 핵심 아키텍처 및 처리 단계**

### **Step 1: 측정 단계 (듣기 및 느끼기)**
* **기능 요약:** 기계의 상태를 사람의 귀와 피부처럼 감지합니다.
* **상세 설명:** 기기 내부에 탑재된 초정밀 센서가 설비의 소음과 미세 진동 데이터를 실시간으로 연속 측정합니다. 거추장스러운 외부 케이블 연결 없이, 설비 표면에 SMEA-100을 부착하는 것만으로 데이터 수집이 시작됩니다.

### **Step 2: 분석 단계 (인간의 청각 모방 필터링)**
* **기능 요약:** 불필요한 잡음을 걸러내고 중요한 소리 특징만 뽑아냅니다.
* **상세 설명:** 수집된 원시 데이터(Raw Data)는 SMEA-100 내부의 프로세서로 전달됩니다. 여기서 '인간의 귀'가 소리를 인식하는 방식을 모방한 특수 알고리즘(음향 특징 추출)이 작동합니다. 공장 내의 일반적인 배경 백색 소음은 무시하고, 기계 결함과 직결되는 특정 주파수 대역의 패턴만을 선별합니다.

### **Step 3: AI 이상 감지 단계 (판단하기)**
* **기능 요약:** 기계가 아픈지, 정상인지 AI가 즉각 판단합니다.
* **상세 설명:** * 기기 내부에 탑재된 초경량 AI(Edge AI) 모델이 Step 2에서 추출된 소리 패턴을 실시간으로 검사합니다.
    * 사전 학습된 정상 상태의 소리 패턴과 현재 소리를 비교하여, 일정 기준을 벗어나는 즉시 **'이상(Anomaly)'** 상태로 판정합니다. 외부 서버로 데이터를 보낼 필요가 없어 지연 시간 없이 즉각적인 판정이 가능합니다.

### **Step 4: 똑똑한 데이터 저장 (기억하기)**
* **기능 요약:** 의미 없는 데이터는 버리고, 문제 상황만 블랙박스처럼 기록합니다.
* **상세 설명:** * "정상 데이터는 저장하지 않습니다." 이것이 SMEA-100의 핵심 데이터 전략입니다.
    * AI가 '이상'으로 판단한 순간을 기준으로, 문제가 발생하기 직전과 직후의 데이터만을 골라냅니다.
    * 이 알짜배기 데이터들은 용량을 최소화하는 '논리적 압축' 기술을 거쳐 기기 내부 메모리에 안전하게 저장됩니다. (마치 자동차 사고가 났을 때만 저장되는 블랙박스와 같습니다.)

### **Step 5: 사용자 인터페이스 및 연동 (알려주기)**
* **기능 요약:** 스마트폰으로 바로 확인하거나, 기존 시스템에 알아서 보고합니다.
* **상세 설명:**
    * **독립 작동 모드 (No Server/App):** SMEA-100 자체가 하나의 작은 웹 서버 역할을 합니다. 사용자는 별도의 앱 설치 없이 스마트폰이나 PC의 웹 브라우저 주소창에 기기의 IP 주소만 입력하면, 직관적인 대시보드 화면을 통해 현재 상태와 과거 이상 기록을 즉시 확인할 수 있습니다.
    * **API 연동 모드:** 사내에 운영 중인 MES(제조실행시스템)나 대형 모니터링 화면이 있다면, SMEA-100에서 제공하는 표준 API를 통해 이상 발생 알림과 압축된 데이터를 자동으로 전송하도록 쉽게 설정할 수 있습니다.
